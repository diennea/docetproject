Sample Documentation Package

A simple documentation package to show how to build comprehensive guides via simple but powerful Docet language.

This is a sample documentation package to show you how easy it is to build comprehensive guides by using simple but powerful Docet language.
Docet easy-to-integrate framework will do the rest, by providing a fast way to integrate documentation packages such as this one, into your Java Web application.

Note
For an example on how integrate Docet into your Java Web app, please have a look at project docet-sample-app
